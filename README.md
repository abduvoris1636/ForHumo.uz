# Humo eSport Tournament Platform

O'zbekistonning eng yirik eSport turnirlarini boshqarish platformasi.

## 🚀 Live Demo
- Website: [https://humoesport.vercel.app](https://humoesport.vercel.app)
- Admin Panel: [https://humoesport.vercel.app/admin](https://humoesport.vercel.app/admin)

## 🛠️ Features

### Public Features
- Jamoa ro'yxatdan o'tish formasi
- Real-time jamoa nomi tekshiruvi
- Cloudinary orqali logotip yuklash
- Telegram bot orqali bildirishnomalar
- Zamonaviy UI/UX dizayn
- Snowfall animatsiyasi

### Admin Features
- JWT asosida himoyalangan admin panel
- Barcha jamoalarni boshqarish
- Statistikalar va dashboard
- Jamoa holatini o'zgartirish (approve/reject)
- Search va filter funksiyalari

## 📦 Tech Stack
- Next.js 14 (App Router)
- MongoDB (Mongoose)
- TailwindCSS
- Cloudinary
- Telegram Bot API
- JWT Authentication
